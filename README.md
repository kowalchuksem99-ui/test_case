# Телефон → ОКВЭД сопоставление

Небольшая утилита для:
- нормализации российского мобильного номера к формату `+79XXXXXXXXX`,
- загрузки актуального `okved.json` по HTTPS,
- поиска кода ОКВЭД с максимальным совпадением по окончанию номера.

---

## Зависимости

- Python 3.10+
- Внешние библиотеки не используются (только стандартная библиотека Python)

---

## Стандарт документации и оформления кода

В решении используется единый стандарт оформления кода и документации:
- документация функций оформлена в соответствии с **PEP 257**,
- аннотации типов соответствуют **PEP 484**,
- стиль и именование кода соответствуют **PEP 8**.

---

## Запуск

```bash
python main.py "+7 (999) 123-45-67"
